# Vite + React + TailwindCSS Profile App

A simple, fully responsive user profile app built with **Vite**, **React**, and **TailwindCSS**.  
It supports light/dark themes and includes a loading button + toast notifications.

## Features

- Built with **Vite + React**
- Styled using **TailwindCSS v4**
- Light & Dark mode support
- Loading indicator on save
